# Python Projects
This repository contains small programs I created while learning Python.  
Each project helped me practice core programming concepts and basic algorithms.

## Projects Included
- **[Craps Game Simulation](games/craps-dice-game.py)** – A simple dice game simulation demonstrating loops, conditionals, and random number generation.  
- **[Caesar Cipher Encryption Tool](cryptography/caesar-cipher.py)** – A program to encrypt and decrypt text using the Caesar cipher, practicing string manipulation and functions.  
- **[Average Test Score Calculator](calculators/test-average.py)** – Calculates the average of user-inputted test scores, reinforcing user input handling and basic arithmetic.
- **[Math Test Calculator](calculators/math-test.py)** – An interactive program that asks for the user’s name, presents 10 addition questions with immediate feedback, and displays the final score out of 10 along with the average.

## Concepts Practiced
- Loops and conditionals
- Functions and modular code
- User input and data validation
- Basic algorithms and problem-solving
